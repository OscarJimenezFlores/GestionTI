La relaci√≥n entre el valor de la informaci√≥n y el coste de obtenerla puede ser analizada utilizando ratios conocidos como el ROI (Return on Investment), que compara los beneficios (valor) con la inversi√≥n y costes en los que se incurren para lograr tales beneficios. > ***ROI = (Beneficio Neto / Inversi√≥n) √ó 100***

 Donde:
- Beneficio Neto = Ingresos generados ‚àí Costos totales

#### Ejemplo:

| Elemento              | Valor / F√≥rmula                          | Explicaci√≥n                                                                 |

#### Pautas generales para evaluar el ROI

    - ROI > 0%: Si el ROI es positivo, significa que la inversi√≥n gener√≥ un retorno superior a lo invertido, es decir, hubo una ganancia. 

    - ROI = 0%: Si el ROI es cero, la inversi√≥n solo permiti√≥ recuperar lo invertido, sin ganancias adicionales.

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

# Funci√≥n para calcular el ROI
def calcular_roi(inversion_inicial, ganancia_obtenida):
    beneficio_neto = ganancia_obtenida - inversion_inicial
    roi = (beneficio_neto / inversion_inicial) * 100
    return roi

# Datos del caso de estudio
inversion_inicial = 100000  # Costo de implementaci√≥n del sistema
ganancia_obtenida = 150000  # Ahorros e ingresos generados por el sistema

# Calcular el ROI
roi = calcular_roi(inversion_inicial, ganancia_obtenida)

# Datos de las tareas del proyecto
data = {
    'Tarea': ['Planificaci√≥n', 'Desarrollo', 'Pruebas', 'Implementaci√≥n', 'Monitoreo'],
    'Duraci√≥n (d√≠as)': [10, 30, 15, 5, 20],
    'Recursos Asignados (personas)': [2, 5, 3, 2, 4],
    'Estado': ['Completada', 'En progreso', 'Pendiente', 'Pendiente', 'Pendiente'],
    'Costo Estimado ($)': [5000, 30000, 15000, 5000, 10000]
}

# Crear un DataFrame con los datos del proyecto
df_proyecto = pd.DataFrame(data)

# Visualizaci√≥n de las tareas del proyecto
plt.figure(figsize=(10, 6))
plt.bar(df_proyecto['Tarea'], df_proyecto['Duraci√≥n (d√≠as)'], color=['green', 'blue', 'orange', 'red', 'purple'])
plt.title('Duraci√≥n de las Tareas del Proyecto de Gesti√≥n')
plt.xlabel('Tareas')
plt.ylabel('Duraci√≥n (d√≠as)')
plt.show()

# Mostrar el ROI calculado
print(f"El ROI de la inversi√≥n en el sistema de gesti√≥n de proyectos es: {roi:.2f}%")

# Mostrar el DataFrame con detalles del proyecto
print("\nDetalles del Proyecto de Gesti√≥n:")
print(df_proyecto)

# An√°lisis adicional: Distribuci√≥n del costo estimado por tarea

plt.pie(df_proyecto['Costo Estimado ($)'], labels=df_proyecto['Tarea'], autopct='%1.1f%%', startangle=90)
plt.title('Distribuci√≥n del Costo Estimado por Tarea')
plt.show()
